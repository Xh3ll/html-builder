# HTML Builder

Visueller HTML-Code-Generator — Elemente per Klick zusammenstellen und fertigen HTML-Code kopieren. Ideal für CMS-Beiträge und Forum-Posts.

## Download

**[⬇ Installer herunterladen (Windows)](https://github.com/Xh3ll/html-builder/releases/latest)**

Die App installiert sich wie gewohnt und erscheint unter Windows in „Apps & Features". Bereits installierte Versionen werden automatisch über Updates benachrichtigt.

## Features

- Elemente per Klick hinzufügen: Div, Absatz, Bild, Überschrift, Tabelle, Liste, Link u.v.m.
- Drag & Drop zum Umstrukturieren
- Tabellen-Editor mit Zell-Zusammenführung (colspan/rowspan)
- HTML importieren und weiterbearbeiten
- Vorschau im Browser
- Projekte speichern & laden (.htmlbuilder)
- Autosave
- Undo/Redo (Strg+Z / Strg+Y)
- Auto-Updates bei neuen Versionen
